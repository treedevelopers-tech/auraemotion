Aura is an advanced **Emotion-Aware AI Chatbot** that detects a user's facial emotion in real-time and adapts chatbot responses accordingly.

It uses a **CNN model trained on the FER2013 dataset** to recognize facial emotions from a webcam feed. The detected emotion is then used to guide responses generated by **Google Gemini 1.5**, enabling emotionally intelligent conversations.
